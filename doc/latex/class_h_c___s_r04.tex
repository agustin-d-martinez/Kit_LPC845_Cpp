\doxysubsection{Detailed Description}
Clase del objeto \doxylink{class_h_c___s_r04}{HC\+\_\+\+SR04} El objeto \doxylink{class_h_c___s_r04}{HC\+\_\+\+SR04} Mide distancia mediante el uso de un ultrasónico. Debido a los tiempos muy pequeños de uso, no se recomienda utilizar en grandes cantidades. 

SENSOR ULTRASÓ\+NICO HC-\/\+SR04\+: Se le debe enviar PWM de 60ms o más (recomendamos 80) con un tiempo de encendido de 10us. Recebirá otro pulso variable que debe ser leído por tx. Devuelve el valor de distancia en cm (centímetros)
